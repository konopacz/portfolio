Game Design Blog

This is a development blog for a 2D platformer made with Unity and C#, begun in 2018. Periodically, updates and features will slowly be added to the game, and shared here. Everything is designed/created by Erik Laats.

Post #1: January 29, 2018

While all artwork is still stand-in artwork, a lot of basic functionality has come together for this first iteration, including basic platformer mechanics and text interactions with a talkative rock.

Also implemented was a fully functional main and pause menu, death mechanics, and a collectible. I also scripted a way to control level availability through collectible count, which can then be saved. This makes it so a player can beat a level and close the game, and still come back and load their progress.

While it may be filled with stand-in animations (and also looks like a mess), the animator is fully built, with (mostly) working transitions for all basic animations.

Stay tuned for concept art and more development steps this month!
