Game Design Blog

This is a development blog for gaming projects made with Unity, C#, and/or Javascript. All assets of all projects are designed/created by Erik Laats.

Leffriend's Gauntlet

"Leffriend's Gauntlet" is a small project designed to use speech recognition with the goal of accessibility. It is made entirely with HTML, CSS, and JS, and took only a few days. You can play the game here.

Platform Mechanics: 1/18/18

This is a practice project to work on platforming mechanics, saving, and other fun things for a future game.

Also implemented was a fully functional main and pause menu, death mechanics, and a collectible. I also scripted a way to control level availability through collectible count, which can then be saved. This makes it so a player can beat a level and close the game, and still come back and load their progress.

While it may be filled with stand-in animations (and also looks like a mess), the animator is fully built, with (mostly) working transitions for all basic animations.

Stay tuned for concept art and more development steps next month!
